# UZT Car Simulation: Mathematically Secured 3D Environment

An interactive 3D simulation exploring the **Ultimate Zero Theory (UZT)**, developed to visualize motion and time at the limit where traditional physics encounter singularities.

## 🌌 The Logic of UZT

In traditional Newtonian physics, velocity is defined as $v = d/t$. As time ($t$) approaches zero, velocity traditionally approaches infinity, creating a mathematical singularity.

**The UZT Axiom:** UZT proposes a fundamental shift in the division by zero:  
> **$m/0 = 0$** (Mass or Distance divided by Zero Time equals Zero).

This simulation demonstrates that if time stops entirely, motion ceases to exist, dropping speed ($v$) to $0$. The object enters a state of absolute rest (referred to as **STEADY**), effectively avoiding impossible infinite velocities.

## 🚀 Features

* **3D UZT Chamber:** A controlled environment to observe an "Energy Packet" under UZT laws.
* **Dynamic Inputs:** Adjust distance and time to see real-time velocity calculations.
* **Scale Toggles:** Switch between **Local vs. Cosmic** scales and **Norm vs. Milli** precision.
* **State Monitoring:** Track the object's status from active motion to the **STEADY** state.
